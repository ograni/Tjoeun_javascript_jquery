JavaScript 문서 정보를 제어하는 Document객체

문서 정보를 표시해주는 속성들

document.title - 문서의 제목을 표시한다
document.location - 문서의 위치 경로를 표시한다

사용예) 

<head>
    <title> Test </title>
    <script>
        document.write("이 홈페이지의 제목은", document.title,"입니다.", "<br>")
        document.write("이 홈페이지의 위치는", document.location, "입니다.")
    </script>
</head>

 

최종 수정 날짜를 표시해주는 속성

document.lastModified

사용예)

<head>
    <title>연습</title>
    <script>
        function modifi()
        {
            a=new Date(document.lastModified)
            x=a.getYear()
            y=a.getMonth()+1
            z=a.getDate()
            document.write("이 홈페이지의 최종 수정 날짜는", x, "년", y, "월", z, "일입니다.")
        }
    </script>
</head>
<body onload="modifi()">
</body>

 

문서 색상 정보를 제어하는 속성들

document.bgColor="색깔" 문서의 배경색을 설정
document.fgColor="색깔" 문서의 글자색을 설정
document.linkColor="색깔" 링크가 설정된 글자의 색을 설정
document.alinkColor="색깔" 링크가 실행될 때의 글자색을 설정
document.vlinkColor="색깔" 링크가 실행된 후의 글자의 색을 설정

창의 내용을 보여주는 메소드들

a.document.open() - 새로운 창에 내용을 나타나게 만든다.
a.document.close() - 새로운 창에 나타나게 할 내용을 종료한다

사용예)

다음 부분을 <body>태그 부분에 삽입한다

<script>
function win()
{
    a=window.open()
    a.document.open()
    a.document.write("저의 집에 오신 것을 환영합니다.")
    a.document.close()
}	

function win_close()
{
    a.document.open()
    a.document.write("")
    a.document.close()
}
</script>
	
다음 부분을 <body>태그 부분에 삽입한다

<form>
    <input type="button"  value="메시지 창 띄우기"  onclick="win()" />
    <input type="button"  value="내용 지우기"  onclick="win_close()" />
</form>

 

사용예) 그림이 순서대로 바뀌는 효과

<img src="fall1.gif name="shot"> - fall1.gif그림에 shot라는 이름을 설정한다.
document.shot.complete - 문서에서 shot 이름을 가지고 있는 그림이 전송 완료되었는지 확인
document.shot.src="fall"+num+".gif" - 그림을 fall2, fall3, fall4의 순서대로 바꿔준다.
setTimeout("timer()",2000); - 재귀적 용법으로 2초마다 time()함수를 실행 

1) 가을동화.html 파일을 불러와 <img> 태그 안에 name 속성을 'shot'로 설정
    <p style="text-align: center;"><img src="fall1.gif" name="shot" border="1"></p>
2) 다음 내용을 <head> 태그 사이에 입력한다
<head>
    <title>가을동화</title>
    <script>
        var num = 2;
        function timer () {
            if(document.shot.complete) {
                document.shot.src = "fall" + num + ".gif";
                num++; if(num > 4) num = 1;
            }
            setTimeout("timer()", 2000);
        }
    </script>
</head>

3) 다음 내용을 <body> 태그 사이에 입력한다 
<body  style="background: #e89828 url(back.gif) no-repeat;"  onLoad="timer();">
